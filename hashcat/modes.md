# Hashcat Modes (-m)

| Mode | Hash Type |
|------|-----------|
| 0    | MD5       |
| 100  | SHA1      |
| 1000 | NTLM      |
| 1400 | SHA256    |
| 1419 | SHA256 ($pass, $salt)  |
| 1800 | SHA512    |
| 3200 | bcrypt    |
| 22000 | WPA-PBKDF2 |
